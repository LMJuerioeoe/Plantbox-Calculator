# 🌱 Istutuskasti suuruse kalkulaator

Veebipõhine tööriist aiamaa **istutuskastide suuruste võrdlemiseks ja visualiseerimiseks**.

👉 Töötab otse brauseris (ei vaja serverit)  
👉 Sobib GitHub Pages jaoks  
👉 Kõik joonised on omavahel **õiges mõõtkavas**

---

## ✨ Funktsioonid

- Muuda **paneeli (külje) pikkust**
- Vali erinevad kastid:
  - 5-paneeline
  - 6-paneeline
  - 8-paneeline (ja rohkem)
- Näed kohe:
  - 📏 välisläbimõõtu
  - 📐 pindala (m²)
  - 📦 mullamahtu (liitrites)
  - 📏 ümbermõõtu
- Visuaalne võrdlus **kõrvuti**
- SVG eksport (nt lõikamiseks / joonestamiseks)
- Prinditav vaade
